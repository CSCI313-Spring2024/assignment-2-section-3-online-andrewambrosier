Portfolio

Capstone Project

My capstone project, working with Razor Tracking in Fargo, involves the development of a system that integrates Unity's Gaming Engine to render digital twins in a 3D virtual environment. This system aims to accurately model real-world objects and environments, enabling users to interact with and analyze them in a virtual space. The project combines cutting-edge game development techniques with practical applications in industries such as logistics, urban planning, and manufacturing.

Cybersecurity Project

My cybersecurity project focuses on developing decision-making software to aid in responding to security incidents. The software analyzes various threat scenarios, evaluates potential responses, and helps guide the decision-making process in real-time. The project aims to streamline incident response, improve efficiency, and minimize risks by automating key decision points and providing actionable insights to security teams.
